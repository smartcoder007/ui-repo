Reference: https://www.npmjs.com/



1) create new node project (add package.json)
    project path>npm init
	
2) add external dependency to the project
    >npm install --save express@4.16.3 mysql@2.15.0
	
3) Run the node project
   >node index.js
   
 4) Automatic detect changes using nodemon
   > npm install -g  nodemon
   running project using nodemon
   >nodemon index.js
   
   
 5)  download dependency from exiting npm project
     project path>npm install
     
 6) Star the server
      >npm start
